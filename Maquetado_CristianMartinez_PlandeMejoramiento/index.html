actividad

Replicar los siguientes 10 Layouts utilizando HTML y CSS, empleando el modelo Flexbox y haciendo uso de elementos semánticos. Se de lograr el mayor parecido posible con la imagen proporcionada.

Se debe entregar el link de un repositorio público en GitHub, el cual debe tener la estructura de archivos que se muestra en la imagen. No debe faltar ningún archivo ni tener otros nombres diferentes, de no ser así se descontarán puntos.

Adicionalmente, todo el repositorio debe contar con un menú principal llamado index.html que permita navegar por los distintos Layouts. El diseño de este menú principal se deja a su gusto.
